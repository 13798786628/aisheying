## 🆘 常见问题快速解决

### 无法访问 5173 端口
```bash
# 检查安全组
访问：https://console.cloud.tencent.com/cvm/securitygroup
添加入站规则：TCP 5173 端口，来源 0.0.0.0/0

# 检查防火墙
firewall-cmd --list-ports
```

### PM2 服务未启动
```bash
pm2 status
pm2 logs wedscene
pm2 restart wedscene
```

### Node.js 版本过低
```bash
node -v
# 如果低于 v18，重新安装
curl -fsSL https://rpm.nodesource.com/setup_18.x | bash -
yum install -y nodejs
```
