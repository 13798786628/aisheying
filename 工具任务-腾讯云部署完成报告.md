## 🌐 域名配置（推荐）

### 为什么需要域名？
- ✅ 专业形象
- ✅ 易于记忆
- ✅ 支持 HTTPS
- ✅ 视频生成需要公网访问

### 配置步骤

1. **域名解析**
   - 添加 A 记录指向服务器 IP

2. **安装 Nginx**
   ```bash
   yum install -y nginx
   ```

3. **配置反向代理**
   ```bash
   cp nginx-config.conf /etc/nginx/conf.d/wedscene.conf
   nano /etc/nginx/conf.d/wedscene.conf  # 修改域名
   nginx -t
   systemctl restart nginx
   ```

4. **配置 SSL**
   ```bash
   certbot --nginx -d your-domain.com
   ```

5. **更新 PUBLIC_BASE_URL**
   ```bash
   nano /www/wwwroot/wedscene-ai/.env
   # PUBLIC_BASE_URL=https://your-domain.com
   pm2 restart wedscene
   ```

## 🐛 常见问题快速解决

### 1. 无法连接服务器
```bash
# 检查腾讯云安全组
登录腾讯云控制台 → 云服务器 → 安全组 → 添加规则
端口: 22, 80, 443, 5173
来源: 0.0.0.0/0
```

### 2. npm install 失败
```bash
npm install --registry=https://registry.npmmirror.com
```

### 3. PM2 启动失败
```bash
pm2 logs wedscene      # 查看错误
node server.mjs        # 手动测试
```

### 4. 视频生成失败
```bash
# 检查配置
cat .env | grep PUBLIC_BASE_URL
# 确认可以从外网访问
curl https://your-domain.com
```
