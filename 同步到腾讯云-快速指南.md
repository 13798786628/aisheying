# 🚀 同步到腾讯云 - 快速上手指南

## 📋 三种部署方式（选择一种）

### 方式 1：使用上传脚本（最简单）⭐

#### Windows 用户
1. 在项目目录右键，选择 "Git Bash Here"
2. 运行上传脚本：
```bash
bash upload-to-server.sh
```
3. 按提示输入服务器 IP 和 SSH 端口
4. 等待上传完成
5. SSH 连接到服务器，运行部署脚本

#### 步骤详解
```bash
# 1. 本地打包上传（在本地 Git Bash）
cd "C:\Users\22591\Desktop\婚礼ai视频"
bash upload-to-server.sh

# 2. SSH 连接到服务器
ssh root@your-server-ip

# 3. 在服务器上部署
cd /www/wwwroot/wedscene-ai
bash deploy-to-server.sh
```

## ⚙️ 服务器配置（首次部署）

### 1. 安装 Node.js
```bash
# CentOS/RHEL
curl -fsSL https://rpm.nodesource.com/setup_18.x | bash -
yum install -y nodejs

# Ubuntu/Debian
curl -fsSL https://deb.nodesource.com/setup_18.x | bash -
apt-get install -y nodejs

# 验证
node -v
npm -v
```

### 2. 安装 PM2
```bash
npm install -g pm2
```

### 3. 安装依赖
```bash
cd /www/wwwroot/wedscene-ai
npm install --production --registry=https://registry.npmmirror.com
```

### 4. 配置环境变量
```bash
cp .env.example .env
nano .env
```

**重要配置**：
```bash
PORT=5173
PUBLIC_BASE_URL=https://your-domain.com
OPENAI_API_KEY=你的密钥
XIAOJI_API_KEY=你的密钥
ADMIN_TOKEN=管理员密码

# 删除或注释掉本地代理
# HTTPS_PROXY=http://127.0.0.1:7897
# HTTP_PROXY=http://127.0.0.1:7897
```

### 5. 启动服务
```bash
pm2 start server.mjs --name wedscene
pm2 save
pm2 startup
```

### 6. 开放防火墙
```bash
firewall-cmd --zone=public --add-port=5173/tcp --permanent
firewall-cmd --reload
```

### 7. 测试访问
```
http://your-server-ip:5173
```

## 🌐 配置域名（推荐）

### 1. 安装 Nginx
```bash
# CentOS
yum install -y nginx

# Ubuntu
apt-get install -y nginx

systemctl start nginx
systemctl enable nginx
```

### 2. 配置反向代理
```bash
nano /etc/nginx/conf.d/wedscene.conf
```

粘贴配置（已为你准备好 nginx-config.conf 文件）：
```nginx
# 见 nginx-config.conf 文件内容
```

### 3. 测试并重启
```bash
nginx -t
systemctl restart nginx
```

### 4. 配置 SSL（免费 HTTPS）
```bash
# 安装 certbot
yum install -y certbot python3-certbot-nginx

# 获取证书
certbot --nginx -d your-domain.com

# 自动续期
certbot renew --dry-run
```

### 5. 更新 PUBLIC_BASE_URL
```bash
nano /www/wwwroot/wedscene-ai/.env
```

修改为：
```bash
PUBLIC_BASE_URL=https://your-domain.com
```

重启服务：
```bash
pm2 restart wedscene
```

## 🐛 常见问题

### 无法访问服务器
- 检查腾讯云安全组设置（需要开放 22、80、443、5173 端口）
- 检查防火墙配置

### npm install 失败
```bash
npm install --registry=https://registry.npmmirror.com
```

### PM2 启动失败
```bash
# 查看详细错误
pm2 logs wedscene

# 手动测试
node server.mjs
```

### 视频生成失败
- 检查 PUBLIC_BASE_URL 配置
- 确认域名可以从外网访问
- 检查 API 密钥
