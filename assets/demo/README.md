# 演示素材说明

当前 1.0 页面只会打包并加载这些轻量素材：

- `upload-bg.webp`
- `product-showcase.webp`
- `product-core-bg.webp`

同目录下的 `jpg`、`png`、`mp4` 文件只作为本地演示源素材或备份素材使用，不进入生产构建，也不会影响上线包体积。

## 替换建议

如果要替换线上视觉素材，优先导出为 `.webp`，并保持文件名不变：

| 文件名 | 用途 | 建议规格 |
|---|---|---|
| `upload-bg.webp` | 上传区域背景 | 1600px 以内，轻微压缩 |
| `product-showcase.webp` | 首页产品展示图 | 1600px 以内，主体清晰 |
| `product-core-bg.webp` | 核心功能背景 | 1600px 以内，色调克制 |

本地演示视频可以继续放在这里，但公测上线前不要依赖它作为页面必需资源。
