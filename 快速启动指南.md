# 🚀 快速启动指南（精简版）

## 第一次使用（详细版）

### 窗口 1：启动内网穿透
```bash
cd "C:\Users\22591\Desktop\婚礼ai视频"
npx localtunnel --port 5173
```
**等待显示 URL，例如**：`your url is: https://abc-xyz-123.loca.lt`  
**复制这个 URL，保持窗口打开！**

### 窗口 2：配置并启动服务器
```bash
cd "C:\Users\22591\Desktop\婚礼ai视频"
notepad .env
```
**找到**：`PUBLIC_BASE_URL=...`  
**改为**：`PUBLIC_BASE_URL=https://abc-xyz-123.loca.lt` （你的实际 URL）  
**保存并关闭**

```bash
node server.mjs
```
**确认看到**：`Motion Video: omni-flash ... (PUBLIC_BASE_URL=https://abc-xyz-123.loca.lt)`  
**保持窗口打开！**

### 访问服务
- **本地**：http://127.0.0.1:5173
- **公网**：https://abc-xyz-123.loca.lt（首次访问点击 Continue）

## ⚠️ 常见错误

### 错误 1：视频生成失败
👉 检查服务器日志是否显示 "mock mode"  
👉 如果是，说明 PUBLIC_BASE_URL 配置不对，重新配置并重启服务器

### 错误 2：公网访问 503 错误
👉 窗口 1 的 localtunnel 可能已停止  
👉 重新启动 localtunnel，更新配置，重启服务器

### 错误 3：URL 每次都变
👉 这是正常的！localtunnel 免费版每次启动会分配新 URL  
👉 每次启动都要更新 .env 并重启服务器
