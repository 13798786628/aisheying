## 🌐 域名配置（可选但推荐）

### 1. 解析域名
在域名管理面板添加 A 记录：
```
记录类型: A
主机记录: @
记录值: your-server-ip
TTL: 600
```

### 2. 配置 Nginx
```bash
# 安装 Nginx
yum install -y nginx
systemctl start nginx
systemctl enable nginx

# 复制配置文件
cp /www/wwwroot/wedscene-ai/nginx-config.conf /etc/nginx/conf.d/wedscene.conf

# 编辑域名
nano /etc/nginx/conf.d/wedscene.conf
# 将 your-domain.com 替换为你的实际域名

# 测试并重启
nginx -t
systemctl restart nginx
```

### 3. 配置 SSL（免费 HTTPS）
```bash
# 安装 certbot
yum install -y certbot python3-certbot-nginx

# 获取证书
certbot --nginx -d your-domain.com -d www.your-domain.com

# 自动续期
certbot renew --dry-run
```

### 4. 更新配置
```bash
nano /www/wwwroot/wedscene-ai/.env
```

修改：
```bash
PUBLIC_BASE_URL=https://your-domain.com
```

重启：
```bash
pm2 restart wedscene
```

## 🐛 常见问题

### 1. 无法连接服务器
**原因**: 腾讯云安全组未开放端口

**解决**:
1. 登录腾讯云控制台
2. 进入云服务器 → 安全组
3. 添加入站规则：
   - 端口: 22, 80, 443, 5173
   - 来源: 0.0.0.0/0

### 2. npm install 失败
**解决**:
```bash
npm install --registry=https://registry.npmmirror.com
```

### 3. PM2 启动失败
**解决**:
```bash
# 查看详细错误
pm2 logs wedscene

# 检查 Node.js 版本
node -v  # 应该是 v18+

# 手动测试
node server.mjs
```

### 4. 视频生成失败
**检查**:
- PUBLIC_BASE_URL 是否配置正确
- 域名是否可以从外网访问
- API 密钥是否有效
