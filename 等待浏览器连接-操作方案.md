## 连接成功后，我会自动帮你：

### 第一步：访问腾讯云控制台
- 打开 https://console.cloud.tencent.com/
- 如果未登录，会引导你登录

### 第二步：检查云服务器状态
- 进入"云服务器 CVM"
- 查看现有服务器列表
- 获取服务器 IP 地址

### 第三步：配置安全组
自动配置入站规则，开放端口：
- 端口 22 (SSH)
- 端口 80 (HTTP)
- 端口 443 (HTTPS)
- 端口 5173 (应用端口)

### 第四步：获取连接信息
帮你记录：
- 服务器公网 IP
- SSH 端口
- 操作系统类型

### 第五步：提供部署命令
根据获取的信息，生成部署命令：
```bash
bash upload-to-server.sh
# 输入服务器 IP: [自动获取的IP]
# 输入端口: 22
```

## 💡 如果扩展连接有问题

### 方案 A：手动操作（最快）
我可以提供详细的操作步骤，你手动在腾讯云控制台操作：

1. **打开控制台**
   - 访问 https://console.cloud.tencent.com/
   - 登录账号

2. **查看服务器**
   - 左侧菜单：云服务器 → 实例
   - 记录公网 IP

3. **配置安全组**
   - 点击实例 → 安全组
   - 添加入站规则：22, 80, 443, 5173

4. **获取密码**
   - 如果忘记密码，可以重置实例密码

5. **运行部署脚本**
   ```bash
   bash upload-to-server.sh
   ```
