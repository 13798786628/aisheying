## ⚠️ 重要提醒

### 1. Localtunnel URL 会变化
每次重启 localtunnel，URL 都会改变。需要：
- 记录新的 URL
- 更新 `.env` 配置
- 重启服务器

### 2. 保持窗口打开
视频生成期间（几分钟），必须保持：
- Localtunnel 窗口打开
- 服务器窗口打开
- 代理软件运行（如果需要）

### 3. 首次访问确认
访问 localtunnel URL 时，首次需要点击"Continue"按钮。

## 🐛 故障排查

### 问题：视频仍然生成失败

**检查步骤**：

1. **确认 localtunnel 运行**
   - 窗口 1 是否还打开？
   - 是否显示 "your url is: ..."？

2. **确认配置正确**
   ```bash
   cat .env | grep PUBLIC_BASE_URL
   # 应显示：PUBLIC_BASE_URL=https://xxx.loca.lt
   ```

3. **确认服务器日志**
   - 窗口 2 中查看
   - 应显示 "Motion Video: omni-flash ... (PUBLIC_BASE_URL=...)"
   - 不应显示 "mock mode"

4. **测试公网访问**
   ```bash
   curl https://你的URL.loca.lt
   # 应返回 HTML 内容，不是 503 错误
   ```

5. **重启服务**
   - Ctrl+C 停止服务器
   - 重新运行 `node server.mjs`
