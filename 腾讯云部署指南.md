## 方案选择

### 方案 1：Git 仓库部署（推荐）⭐
- **优点**：版本控制、易于更新、支持回滚
- **适合**：有 GitHub/Gitee 账号，需要版本管理
- **时间**：首次 20 分钟，后续更新 5 分钟

### 方案 2：SFTP 直接上传
- **优点**：简单直接、不需要 Git
- **适合**：快速部署、一次性上传
- **时间**：15-30 分钟

### 方案 3：打包上传
- **优点**：上传速度快
- **适合**：网络不稳定时
- **时间**：10-15 分钟

## Git 方式部署（推荐）

### 步骤 1：初始化 Git 仓库

#### 1.1 检查 Git 状态
```bash
cd "C:\Users\22591\Desktop\婚礼ai视频"
git status
```

#### 1.2 初始化仓库（如果还未初始化）
```bash
git init
```

#### 1.3 创建 .gitignore 文件
```bash
# 已存在 .gitignore，检查内容
cat .gitignore
```

### 步骤 2：创建远程仓库

#### 选项 A：使用 GitHub
1. 访问 https://github.com/new
2. 创建新仓库，例如 `wedscene-ai`
3. 选择 **Private**（私有仓库）
4. 不要初始化 README

#### 选项 B：使用 Gitee（国内速度快）
1. 访问 https://gitee.com/projects/new
2. 创建新仓库，例如 `wedscene-ai`
3. 选择 **私有**
4. 不要初始化 README

### 步骤 3：推送代码

#### 3.1 添加文件到 Git
```bash
git add .
```

#### 3.2 提交
```bash
git commit -m "初始提交：婚礼AI视频生成系统"
```

#### 3.3 添加远程仓库
```bash
# GitHub（将 YOUR_USERNAME 替换为你的用户名）
git remote add origin https://github.com/YOUR_USERNAME/wedscene-ai.git

# 或 Gitee
git remote add origin https://gitee.com/YOUR_USERNAME/wedscene-ai.git
```

#### 3.4 推送代码
```bash
git branch -M main
git push -u origin main
```

**提示**：首次推送需要输入用户名和密码（或使用 Personal Access Token）

### 步骤 4：在腾讯云服务器上拉取代码

#### 4.1 SSH 连接到服务器
```bash
# Windows 用户使用 PowerShell 或 PuTTY
ssh root@your-server-ip

# 输入服务器密码
```

#### 4.2 安装 Git（如果未安装）
```bash
# CentOS/RHEL
yum install -y git

# Ubuntu/Debian
apt-get update
apt-get install -y git
```

#### 4.3 克隆项目
```bash
cd /www/wwwroot
git clone https://github.com/YOUR_USERNAME/wedscene-ai.git
cd wedscene-ai
```

**如果仓库是私有的**：
```bash
# GitHub - 使用 Personal Access Token
git clone https://YOUR_TOKEN@github.com/YOUR_USERNAME/wedscene-ai.git

# Gitee - 使用账号密码或 Token
git clone https://YOUR_USERNAME:YOUR_PASSWORD@gitee.com/YOUR_USERNAME/wedscene-ai.git
```

#### 4.4 配置环境变量
```bash
# 复制配置文件模板
cp .env.example .env

# 编辑配置文件
nano .env
# 或
vi .env
```

**重要配置项**：
```bash
PORT=5173
PUBLIC_BASE_URL=https://your-domain.com
OPENAI_API_KEY=你的API密钥
XIAOJI_API_KEY=你的API密钥
# 注释掉本地代理配置
# HTTPS_PROXY=http://127.0.0.1:7897
# HTTP_PROXY=http://127.0.0.1:7897
```

按 `Ctrl + X`，然后 `Y`，回车保存。

### 步骤 5：后续更新代码

#### 本地更新后推送
```bash
# 本地
git add .
git commit -m "更新说明"
git push
```

#### 服务器上拉取更新

## SFTP 方式部署

### 方案 1：使用 FileZilla（图形化，推荐）

#### 1.1 下载 FileZilla Client
https://filezilla-project.org/download.php?type=client

#### 1.2 连接到服务器
```
主机: sftp://your-server-ip
用户名: root
密码: 你的服务器密码
端口: 22
```

#### 1.3 上传文件
1. 左侧窗口：浏览到 `C:\Users\22591\Desktop\婚礼ai视频`
2. 右侧窗口：进入 `/www/wwwroot/`
3. 创建文件夹 `wedscene-ai`
4. 选择所有文件，右键"上传"

**注意**：不要上传以下文件/文件夹
- `.env`（服务器上单独配置）
- `node_modules/`（服务器上重新安装）
- `*.log` 日志文件
- `.cache/` 缓存文件
- `output/` 输出文件

#### 1.4 配置 .env
在服务器上创建 `.env` 文件，配置生产环境参数。

### 方案 2：使用 WinSCP

#### 2.1 下载 WinSCP
https://winscp.net/eng/download.php

#### 2.2 连接设置
```
文件协议: SFTP
主机名: your-server-ip
端口: 22
用户名: root
密码: 你的服务器密码
```

#### 2.3 上传文件
拖拽文件到右侧窗口即可。

### 方案 3：使用 SCP 命令

```bash
# Windows PowerShell 或 Git Bash
cd "C:\Users\22591\Desktop"

# 压缩项目（排除不需要的文件）
tar -czf wedscene-ai.tar.gz \
  --exclude='node_modules' \
  --exclude='.cache' \
  --exclude='output' \
  --exclude='*.log' \
  --exclude='.env' \
  "婚礼ai视频"

# 上传到服务器
scp wedscene-ai.tar.gz root@your-server-ip:/www/wwwroot/

# SSH 连接到服务器
ssh root@your-server-ip

# 解压
cd /www/wwwroot
tar -xzf wedscene-ai.tar.gz
mv 婚礼ai视频 wedscene-ai
rm wedscene-ai.tar.gz
```

## 域名配置

### 方案 1：直接使用 IP + 端口
```
http://your-server-ip:5173
```

**缺点**：需要记住端口，不够专业

### 方案 2：使用 Nginx 反向代理（推荐）

#### 2.1 安装 Nginx
```bash
# CentOS/RHEL
yum install -y nginx

# Ubuntu/Debian
apt-get install -y nginx

# 启动 Nginx
systemctl start nginx
systemctl enable nginx
```

#### 2.2 配置反向代理
```bash
nano /etc/nginx/conf.d/wedscene.conf
```

**配置内容**：
```nginx
server {
    listen 80;
    server_name your-domain.com www.your-domain.com;
    
    # 客户端上传大小限制（视频文件可能较大）
    client_max_body_size 100M;
    
    location / {
        proxy_pass http://localhost:5173;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_cache_bypass $http_upgrade;
        
        # 超时设置（视频生成可能需要较长时间）
        proxy_connect_timeout 600s;
        proxy_send_timeout 600s;
        proxy_read_timeout 600s;
    }
}
```

保存并退出。

#### 2.3 测试并重启 Nginx
```bash
# 测试配置
nginx -t

# 重启 Nginx
systemctl restart nginx
```

#### 2.4 配置 SSL 证书（推荐）

**使用 Let's Encrypt 免费证书**：
```bash
# 安装 certbot
yum install -y certbot python3-certbot-nginx
# 或
apt-get install -y certbot python3-certbot-nginx

# 获取证书
certbot --nginx -d your-domain.com -d www.your-domain.com

# 自动续期
certbot renew --dry-run
```

Certbot 会自动修改 Nginx 配置，添加 HTTPS 支持。

#### 2.5 更新 .env 配置
```bash
nano /www/wwwroot/wedscene-ai/.env
```

修改：
```bash
PUBLIC_BASE_URL=https://your-domain.com
```

重启服务：
```bash
pm2 restart wedscene
```

## 常见问题

### 问题 1：npm install 失败

**解决方法**：
```bash
# 使用淘宝镜像
npm install --registry=https://registry.npmmirror.com

# 或设置全局镜像
npm config set registry https://registry.npmmirror.com
```

### 问题 2：端口被占用

**解决方法**：
```bash
# 查看端口占用
netstat -antp | grep 5173

# 杀死进程
kill -9 进程ID

# 或修改 .env 中的 PORT 配置
```

### 问题 3：PM2 启动失败

**解决方法**：
```bash
# 查看错误日志
pm2 logs wedscene

# 检查 Node.js 版本
node -v  # 应该是 v16+ 或 v18+

# 手动测试启动
node server.mjs
```

### 问题 4：无法外网访问

**解决方法**：
1. 检查防火墙配置
2. 检查腾讯云安全组设置
3. 检查服务是否监听 0.0.0.0（不是 127.0.0.1）

### 问题 5：视频生成失败

**解决方法**：
1. 检查 PUBLIC_BASE_URL 配置
2. 确认域名可以从外网访问
3. 检查 API 密钥配置
4. 查看服务器日志
