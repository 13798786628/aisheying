# WedScene AI 1.0 真实流程测试清单

最后测试时间：2026-05-30

## 已通过

1. 本地服务健康检查
   - `/api/health` 正常返回。
   - 生图提供方、图片模型、文案模型均已连接。

2. 构建和发布自检
   - `npm.cmd run build` 通过。
   - `npm.cmd run preflight` 通过。
   - 生产构建只打包页面必需资源，当前 `dist` 为 15 个文件，约 16.24MB。
   - `dist` 体积主要来自两个正式演示视频：`assets/motion-demo.mp4` 和 `assets/motion-demo-showcase.mp4`。

3. 独立标题文案流程
   - 打开演示区。
   - 使用示例图。
   - 切换到“只写标题文案”。
   - 点击“生成标题文案”。
   - 生成完成后自动保存到“我的资源”。

4. 客户资源库
   - 只展示有图片或视频素材的资源。
   - 文案型资源不会出现在客户资源库。
   - 不展示“复制文案”和“下载发布包”。
   - 图片资源能逐张保存到本地。
   - 单图保存接口可下载真实图片文件。

5. 页面导航
   - `#products` 产品矩阵存在。
   - `#demo` 在线演示存在。
   - 顶部菜单切换独立页面，不再一路长滚动。
   - `#resources` 客户资源库存在。
   - `#launch` 公测说明存在。
   - `#faq` 常见问题存在。
   - 价格套餐、账号系统、点数规划不展示在客户端。

6. 公测访问码
   - 默认不设置访问码时，不影响本地使用。
   - 设置 `PUBLIC_ACCESS_CODE` 后，未登录访问资源接口会返回 401。
   - 输入正确访问码后，资源区恢复显示。

## 上线前还要复测

1. 用真实客户婚礼图跑一次“电影感分镜图”，重点看 6 张是否重复、是否穿帮、是否保持现场色调。
2. 用真实客户婚礼图跑一次“类似婚礼”，重点看 6 张是否属于同一婚礼类别、比例是否跟上传图一致。
3. 在客户常用浏览器里测试下载：Chrome、Edge、微信内置浏览器。
4. 云服务器部署后重新跑：
   - `/api/health`
   - 上传图片
   - 生成标题文案
   - 客户资源库逐张保存图片
5. 公测前确认 API 额度、点数扣费规则和失败补偿规则。
